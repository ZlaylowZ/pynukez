# Contributing

## Setup

```bash
git clone https://github.com/nukez/pynukez.git
cd pynukez
pip install -e ".[dev]"
```

## Tests

```bash
pytest
```

## Code Style

```bash
black pynukez/
isort pynukez/
```

## Pull Requests

1. Fork the repo
2. Create a branch
3. Make your changes
4. Run tests
5. Submit PR

Keep changes small and focused.
